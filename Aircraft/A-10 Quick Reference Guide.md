# A-10 Quick Reference Guide

[A10 Takeoff Tutorial](https://youtu.be/GR_axWOIwyk)

### Pre-Flight

- Brakes Set
- Trim: `15%`
- Flaps: `7º`

### Takeoff

- Set Power: `90%` and hold
- Brakes released
- Set Power: `100%`
- `7-10ª` pitch for climb until...

## *Positive Rate*

- Landing Gear: `UP`
- Flaps: `UP`

### Climb

- `10º` pitch
- `210 kts`

### Cruise

- 10,000 ft. - FL250
- 210-300 kts

---

[A10 Landing Tutorial](https://youtu.be/SjQzu-PmDK4)

### Approach

- Safe to lower gear and flaps below `200 kts`
- Flaps: `7º`
- Pitch: `3-5º`
- Gear down

### Final Approach

- Flaps: `20º`
- `130 kts`

### Landing

- Power set to idle
- Target touchdown speed: `120 kts`
